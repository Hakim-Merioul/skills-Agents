# Skills

A collection of Claude Code skills I use and share publicly.

## Installation

Clone or copy any skill folder into your `~/.claude/skills/`:

```bash
git clone https://github.com/<user>/skills.git
cp -r skills/transposition ~/.claude/skills/
```

Or copy a single file if you prefer manual install.

## Skills

### transposition
Transfer the context of the current conversation into another window —
another Claude Code session, ChatGPT, Claude.ai, Gemini, or a human teammate.
Outputs a markdown file and a copy-paste-ready block. Multi-domain: works
for code, writing, research, brainstorming.
